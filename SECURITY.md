# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please email security@runtimefence.com

Do NOT create a public GitHub issue for security vulnerabilities.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | Yes       |

## Security Measures

- All commits scanned for secrets
- CI/CD includes vulnerability scanning
- Pre-commit hooks prevent accidental secret commits
